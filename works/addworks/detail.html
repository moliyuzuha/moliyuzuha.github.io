overview

Webサイト向け画像・書籍ビューア設置スクリプトです。

中身を編集するのは1ファイルだけ！フォルダを複製してフォルダ名変更&画像をフォルダ内にアップロードするだけで、閲覧画面を自動生成します。

サイトの情報やヘッダー・フッターのリンク設定、カスタマイズなどは、初期設定ファイルを編集するだけで全ページに反映します。

カスタマイズ用のcssファイルを編集することで、自由にご自身のWebサイトに合わせた外観にすることができます。

※画像が重い場合など、読み込み時に一瞬画面崩れが発生する場合があります。気になる場合は、読み込みのタイミングでロード画面を挟むのがおすすめです。

file organization

file name content
work/ このファイルごとWebサイトのルートディレクトリ(index.htmlがある階層)に設置してください。
AddWorks/ スクリプト本体。このフォルダ内のファイルは基本的に編集しないでください。わかる人は自己責任で自由に改造して大丈夫です。
settings/ 設定用フォルダ。編集するのは基本的にこのフォルダ内のファイルだけです。
settings/custom.css スクリプトのCSSを編集したいときは、意図せぬエラーを回避するため、ここに追加する形で記述してください。
settings/sitedata.js サイト全体の設定用ファイル。metaタグやファビコン、ヘッダー、フッターのカスタマイズはここでできます。
settings/workmanager.js 作品データファイル。作品を追加するたびに必要事項を追記すると、自動でビューアを生成します。
230101/ & 230102/ 作品ファイル。作品の日付をYYMMDD形式でフォルダ名に設定します。work/YYMMDD/view.htmlがビューアのURLになります。view.htmlはコピーするだけ。変更するのはフォルダ名と、imgフォルダへの画像追加のみです。あとはworkmanager.jsファイル内のデータを元にview.htmlを生成します。
